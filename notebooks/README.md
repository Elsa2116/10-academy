# Notebooks

Run the notebooks in order:

1. `01_eda_news.ipynb` - Task 1 exploratory data analysis.
2. `02_quantitative_indicators.ipynb` - Task 2 technical indicators.
3. `03_sentiment_correlation.ipynb` - Task 3 sentiment and return correlation.

Place the financial news CSV in `data/raw/` with columns:
`headline`, `url`, `publisher`, `date`, `stock`.

Place historical stock CSV files in `data/raw/prices/` or use the notebook's
`yfinance` download example.
