# Raw Data

Place source datasets here before running the notebooks.

Expected files:

- `data/raw/fnspid_news.csv` for the FNSPID financial news dataset.
- `data/raw/prices/<TICKER>.csv` for historical OHLCV stock price files, such as `AAPL.csv`, `MSFT.csv`, or `GOOGL.csv`.

The raw data files themselves are ignored by Git to avoid committing large or private datasets.
